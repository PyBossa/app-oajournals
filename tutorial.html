Open Access Licenses Journal tutorial

Hi! Welcome to the application tutorial. This tutorial will show you how can add which licenses each journal supports in this Crowdcrafting.org app.

The application is really simple. It loads the name and publisher of a Journal and asks you this question: which open license is using this Journal?

You will have several options, and all you have to do is to Google the journal and get this information.

You will have also to add, if possible, who owns the copyright of the paper.

All you have to do is to select one of the possible answers and you will be done.
